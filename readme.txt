== Installation ==

WordPress:

1. Copy the `country-detection-polylang` folder into your `wp-content/plugins` folder
2. Activate the country-detection-polylang plugin via the plugins admin page
3. Optional: add this on your .env without double comma "POLYLANG_IPLOCATE_API_KEY". Set your iplocate api key like this POLYLANG_IPLOCATE_API_KEY=12345abcde.

How to get your API Key from IPLocate:
Visit this link - https://www.iplocate.io/
